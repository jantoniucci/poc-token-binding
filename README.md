# poc-token-binding

Naif PoC on accessing TLS session id to be used for token binding

## Getting Started

1. Clone the repo
```
$ git clone https://github.com/jantoniucci/poc-token-binding.git
```
2. Launch the server
```
$ cd poc-token-binding.git
$ ./1.start_server.sh
```
3. Launch a browser
```
$ ./2.start_browser.sh
```

### Prerequisites

NodeJS v8.9+
